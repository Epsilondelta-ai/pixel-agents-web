# Pixel Agents Web

**[English](README.en.md) | [한국어](README.ko.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [Español](README.es.md) | [Português (BR)](README.pt-BR.md) | [Français](README.fr.md) | [Русский](README.ru.md) | [Deutsch](README.de.md)**

Автономная веб-версия [Pixel Agents](https://github.com/pablodelucca/pixel-agents) — анимированный пиксель-арт офис, в котором ИИ-агенты для программирования оживают в виде персонажей.

![Скриншот Pixel Agents](../public/Screenshot.jpg)

## Возможности

- **Анимированные пиксельные персонажи** — каждый агент получает своего персонажа с анимациями ходьбы, набора текста и чтения
- **Отслеживание активности в реальном времени** — персонажи анимируются в зависимости от действий агента (написание кода, чтение файлов, выполнение команд)
- **Редактор планировки офиса** — проектируйте свой офис с полами, стенами и мебелью с помощью встроенного редактора
- **Облачка с текстом** — визуальные индикаторы, когда агент ожидает ввода или нуждается в разрешении
- **Звуковые уведомления** — необязательный сигнал, когда агент завершает свой ход
- **Визуализация субагентов** — субагенты появляются как отдельные персонажи, связанные с родителем
- **Сохранение планировок** — дизайн вашего офиса сохраняется в localStorage
- **Разнообразные персонажи** — 6 разнообразных персонажей на основе [JIK-A-4, Metro City](https://jik-a-4.itch.io/metrocity-free-topdown-character-pack)
- **Без VSCode** — работает полностью в веб-браузере

## Быстрый Старт

```bash
git clone https://github.com/pablodelucca/pixel-agents-web.git
cd pixel-agents-web
npm install
npm run dev
```

Затем откройте `http://localhost:5173` в браузере.

## Использование

1. Откройте приложение в веб-браузере
2. Нажмите **+ Agent**, чтобы создать демо-персонажа
3. Наблюдайте, как персонаж анимируется с имитацией программирования
4. Нажмите **Layout**, чтобы открыть редактор офиса и настроить пространство
5. Нажмите **Settings** для экспорта/импорта планировок и настройки звука

## Сборка для Продакшена

```bash
npm run build
npm run preview
```

Собранные файлы находятся в `dist/` — разверните на любом статическом хостинге (Netlify, Vercel, GitHub Pages и т.д.).

## Технологический Стек

- **React 19** с TypeScript
- **Vite** для разработки и сборки
- **Canvas 2D** для попиксельно-точного рендеринга
- **BFS поиск пути** для передвижения персонажей
- **Конечный автомат персонажей** (ожидание → ходьба → набор/чтение)

## Как Это Работает

Этот проект — автономный веб-порт [расширения Pixel Agents для VSCode](https://github.com/pablodelucca/pixel-agents). Оригинальное расширение отслеживает JSONL файлы транскрипций Claude Code для мониторинга активности агентов. Эта веб-версия:

- Загружает все ассеты (спрайты персонажей, тайлы стен, тайлы полов) непосредственно в браузере через Canvas API
- Заменяет мост сообщений расширения VSCode автономным веб-мостом
- Предоставляет демо-систему симуляции, генерирующую реалистичную активность агентов
- Сохраняет 100% идентичный рендеринг Canvas 2D с оригиналом
